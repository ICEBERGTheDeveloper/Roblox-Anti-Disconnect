## Installation
* First, download the most recent version of the Anti-Disconnect Macro through the most recent GitHub Release (Download Anti-Disconnect.zip)
* After downloading, extract the ZIP file to your desired directory.

**After installation, you can run the "Anti-Disconnect.exe" file to start the macro!**
## Before starting the macro:

* ensure that you extracted the zip file
  - The macro can't update without it!
* ensure that you have selected at least one action

## Features
Anti-Disconnect has lots of features to keep you in game while you go afk, these are the following:
* Resets your character
* Moves your character forward and then backward again
* Moves your camera left and right
* Says custom messages in the chat!
* Select one of the options above, and choose how often it does that thing!
* Automatically updates when a new release comes out
